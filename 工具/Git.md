## 定位
- Git 是分布式版本（repository）控制系统，每台电脑上都是完整的版本库，所以多人协作事不需要联网也可以，只需要把自己的修改推送给对方，然后进行合并
- Git 可以追踪每个文件的修改，删除<br/>
